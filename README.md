## Run Armour
- Linux
```commandline
python3.9 -m venv ArmourVenv
source ArmourVenv/bin/activate
pip install pyAesCrypt PyQt5 zxcvbn pyotp qdarkstyle
cd src
python3.9 main.py
```
- Windows
```commandline
python3.9 -m venv ArmourVenv
ArmourVenv\Scripts\activate.bat
pip install pyAesCrypt PyQt5 zxcvbn pyotp qdarkstyle
cd src
python3.9 main.py
```
